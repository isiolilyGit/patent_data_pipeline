PROJECT DESCRIPTION

This project is a complete end-to-end patent analytics
data engineering pipeline developed using:

- Python
- pandas
- PostgreSQL
- SQL
- Streamlit
- Plotly
- NetworkX

The pipeline performs:

1. Reading raw TSV patent datasets
2. Cleaning and transforming data
3. Loading into PostgreSQL
4. Running analytical SQL queries
5. Generating reports
6. Creating interactive visualizations
7. Building an interactive dashboard
